Razones por las que realicé la página web

Sobre mi

Soy un estudiante de Desarrollo de Aplicaciones Web en primer año y como proyecto final de la asignatura de Lenguaje de Marcas (LND), decidí llevar a cabo la creación de esta pequeña página web, de caracter estático, pero con algunos elementos de JavaScript que al menos me permitan dar un mínimo de dinamismo a la página web.


El tema de la página

Desde mi punto de vista era un tema muy interesante para una página web, incluso de caracter estático, una relacionada con el tiempo meteorológico, ya que suelen ser páginas con multiples campos a lo largo de la página y que suelen ser diferentes en general, aportándome así un recorrido bastante completo en la realización de las diferentes partes que puede contener cualquier página web.


Cosas que podría realizar en el futuro

  • Añadir el responsive a la página web.
  • Añadir el tiempo meteorológico en tiempo real.
  • Implementar un buscador de lugares para que se muestren en los datos del tiempo.
